# TabPFN-based-FRP-tensile-strength-retention-prediction-platform-
To run the Prediction platform, please ensure that:
1. Python is installed in the computer.
2. Necessary libraries are installed in the computer, which are streamlit, pandas, numpy, sklearn, tabpfn, matplotlib and pickle.
To quick install those libraries, the following steps are recommended.
1): Enter Win+R
2): Type in "cmd" and start
3): Type in "pip install ###" (### is the corresponding library, for example, pip install streamlit)

To initialize the platform, please follow those steps:
1): Enter Win+R
2): Type in "cmd" and start
3): Extrat the files into one folder and copy the folder path
4): In the cmd window, type in "cd /d ###" (### is the copied folder path, for example cd /d D:\Prediction_platform)
5): Type in "streamlit run GUI.py"
6): Platform initilized
